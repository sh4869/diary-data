# コンポーネントテスト

```css
.day {
  display: block;
  text-align: center;
  margin-bottom: 1rem;
}

.day a {
  text-decoration: none;
  color: #111;
  display:inline-block;
}

.day h2 {
  text-decoration: none;
  color: #111;
  margin-top: 0.4rem;
  margin-bottom: 0.8rem;
  display: inline-block;
}

.day div a h2:hover {
  text-decoration: underline;
}
```
